🏁 Finish Line
How the Finish Works
No stop-the-room moment. The Greylag Run ends the same way it was raced — every table completes their turn. The Finish Straight (tile 3-5, Velocity, DC 10) is called like any other tile. When the round ends, call standings. If two or more tables cleared in the same round, run the photo finish roll-off first.
⚙  After any finish result — two seconds of room reaction, then continue turn order. Do not hold the round for extended narration.
Finish Straight — Result Calls
🏁 Blazing Run Beat DC by 5+ Call it to the room. Give it a line. Mark tracker. Keep going in turn order.
✓ Clean Finish Beat DC Name the crew. Give it a line. Move on.
✗ Miss Fail DC — roll again next round One line — they roll again next round. DC 10, won’t last long.
Photo Finish — Two or More Tables Clear Same Round
Finish the round first. Once all tables have rolled, announce the dead heat. Each tied table’s chosen roller picks up one d20. Roll simultaneously. Highest result wins. Re-roll ties until broken. No modifiers — raw dice only. Then proceed to full standings.
⚙  If a tied table had a Blazing Run on the Finish Straight, call that out when announcing the tie — not during the roll-off.
Calling Standings
1st place Call it with weight. Name the crew, name the construct. One specific line on their race if you have it.
2nd / 3rd Name and position. One line if warranted.
4th – last Name and position. Keep pace. Don’t skip anyone.
Last place Give it a line. Never a joke at their expense.
