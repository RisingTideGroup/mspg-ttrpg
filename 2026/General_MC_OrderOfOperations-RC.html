Walk every table before doors open.
  • Dice set present — at least one d20 per player, extras for group rolls

GM briefing points:
  • Walk GM Advisor Brief and Table DM Brief
  • Confirm they know the Phase 2 handoff signal
  • Remind: prep crew for team name, construct drawing, sponsor pitch

GRID ORDER ROLL
After all sponsor pitches, run initiative roll before calling the break.
MC Line
"Before we break — we need to set the starting grid. Every player at every table picks up a d20. On my count — three, two, one — roll. GMs: add up your results and divide by player count, round down. That's your grid score. Call it out."
  • All tables roll simultaneously on MC's count
  • Each GM sums results ÷ player count (round down)
  • GMs call out their score — MC and second MC note the order
  • Ties: tied tables re-roll between themselves only — single die, highest wins
  • MC announces Round 1 grid order to the room
Round 1 Only
This roll sets the starting grid and Round 1 turn order only. From Round 2 onward, turn order is always full reverse grid: last place goes first, first place goes last.
BREAK — MC FINAL PREP
  • MC calls the break — players refresh, grab drinks, stretch
During the break:
  • Jonathan enters team names, table numbers, grid order into PowerPoint. Uploads whiteboard photo for each table.
  • Jonathan enters grid order into room tracker: team names, sponsor tier, grid position — hidden until Phase 3 reveal
  • PowerPoint slide order locked last to first for grid reveal
  • Course map queued on projector ready for Phase 3
  • F1-style intro video cued
  • Jonathan and Dan final notes before handoff
Handoff Note
Phase 2 ends here. Course map reveal, race briefing, and Phase 3 launch all happen after the break at the top of Phase 3.

PHASE 3 OPENING — COURSE REVEAL
  • Jonathan welcomes the room back from the break
  • Course map displayed on projector — 60–90 seconds of silent viewing time
  • MC gives one brief contextual comment — no rules yet. Let the room look.
  • GMs hand out course layout cards (no DCs) to their table now
RULES BRIEFING (JONATHAN + DAN)
  • Turn order — single roller per table; last place goes first from Round 2 onward
  • Sponsor use declared room-wide before any dice are picked up each round
  • Boost — fires when last-place table is 3+ tiles behind the next car ahead; MC announces it
  • Battles — triggered when two or more tables share a tile at end of round
  • Natural 1 rules — bottom half: Boost eligible next round; top half: skip next turn
  • Each tile is Easy / Standard / Hard — GMs know exact DCs. All tables roll simultaneously; GMs hold results until called.
  • Jonathan: "Any questions? GMs have all the knowledge." → hand to Dan → F1 intro video → King of the Kingdom sounds the starting signal.
Round Timing
Each round targets ~5 min. Early rounds 7–8 min as the room finds rhythm. Mid-race ~5 min. Late race ~4 min. Full Phase 3 averages ~76 minutes.
EACH ROUND — STEP BY STEP
1
Open the Round
Announce Round [N]. Check Boost eligibility (3+ tile gap, last place only). Announce Boost publicly before any dice are touched.
2
Boost Resolution (if active)
Boost recipient rolls big d6 and selects Boost card result out loud. MC announces to room. If Boost requires room-wide roll: call everyone to roll d20, build to the top result for maximum drama.
3
Resolve Battles
If any tables share a tile from the previous round, resolve before new rolls. Back of pack to front. Each tied table's roller: one d20 simultaneously; highest advances one extra tile. Ties re-roll.
4
Sponsor Window
MC: "Sponsor window is open. Declare any uses now — before anyone picks up a die." All declarations out loud. Uses expended on declaration regardless of outcome.
5
All Tables Roll
MC: "OK — GMs take it away." Allow 1–2 minutes. All tables hold results — no movement yet.
6
MC Calls Results
Last-to-first (reverse grid). Call the table → GM announces roll + modifiers → MC resolves outcome with flavour narration (30 sec max) → move token → call next table.
7
Update Tracker
Click Next Turn on room tracker. Re-order by new grid position. Set Boost eligibility. Open Round [N+1].
Boost Rules — Late Race
If the leader has not crossed the finish by Round 16, MC may call Boost Rules — all DCs drop by 2. MC discretion; Round 16 is the minimum threshold, not automatic. Fires in ~23% of races.
FINISH LINE — HOW IT WORKS
No stop-the-room moment. The round completes normally — every table rolls in turn order. The Finish Straight (tile 3-5, Velocity, DC 10) is called like any other tile. Standings are called after the round ends, not mid-round.

PHOTO FINISH — TWO OR MORE CLEAR THE SAME ROUND
  • Finish the round — all remaining tables complete their turns
  • MC announces the dead heat to the room
  • Each tied table's chosen roller picks up one d20 — roll simultaneously — highest result wins — re-roll ties until broken — no modifiers, raw dice only
  • If a tied table had a Blazing Run, call that out when announcing the tie — not during the roll-off
  • Proceed to standings
